# 第三方项目与许可证说明

本仓库只提供编排工作流、启动器和辅助代码，不重新分发下列项目的源代码、二进制文件、模型权重或声库。

| 项目 | 官方仓库 | 本工作流中的作用 |
|---|---|---|
| Applio | <https://github.com/IAHispano/Applio> | RVC 训练、特征提取、音高跟踪和声音转换 |
| GPT-SoVITS | <https://github.com/RVC-Boss/GPT-SoVITS> | UVR5 / BS-RoFormer 人声伴奏分离 |
| OpenUtau | <https://github.com/openutau/OpenUtau> | MIDI、歌词、音高曲线和演唱表情编辑 |
| DiffSinger | <https://github.com/openvpi/DiffSinger> | 可控歌声合成与 AI 虚拟歌手声库 |
| YuE | <https://github.com/multimodal-art-projection/YuE> | 从歌词生成完整歌曲草案 |
| AudioCraft / MusicGen | <https://github.com/facebookresearch/audiocraft> | 文本或参考旋律生成音乐草图 |

## 使用前必须检查

1. 项目代码许可证。
2. 模型权重许可证。
3. 下载的第三方声库许可证。
4. 数据集和样本的使用范围。
5. 商业使用、署名和再分发要求。

代码开源不等于所有模型、声库和生成结果都自动允许商用。许可证可能随版本变化，请以各项目当前仓库和模型页面为准。
