# 训练与 AI 翻唱

## 训练阶段

工作台调用 Applio 完成以下步骤：

1. 将多段录音统一格式并切成短片段。
2. 使用 RMVPE 提取基频和音高轨迹。
3. 使用 ContentVec 提取发音与内容特征。
4. 训练 RVC 声音模型。
5. 生成 FAISS 检索索引。
6. 将最终 `.pth` 与 `.index` 导出到 `data/models/<模型名>`。

首次建议使用 300 轮。如果结果已经稳定，不需要一味增加轮数。过度训练可能让模型记住录音噪声。

## 翻唱阶段

```text
完整歌曲
  -> BS-RoFormer 分离
  -> 原唱干声 + 伴奏
  -> RMVPE 跟踪原唱音高
  -> RVC 替换为目标音色
  -> 与伴奏混合
  -> WAV + MP3
```

## 参数解释

### 升降调

- `0`：保留原唱音高。
- `+12`：升高一个八度。
- `-12`：降低一个八度。
- 男声替换女声时常需要尝试 `-12`，女声替换男声常需要尝试 `+12`，但具体取决于模型音域。

### 人声音量

- 默认 `1.0`。
- 人声被伴奏盖住时提高到 `1.05` 至 `1.20`。
- 人声刺耳或突出时降低到 `0.80` 至 `0.95`。

### 轻度音准修正

它适合修正小偏差，不会重新创作旋律。原唱本身音准良好时通常不需要开启。

## 质量问题

- 和声很重：分离器可能把多人声音放在同一轨。
- 强混响：换声后可能保留原歌手空间残响。
- 极高音：目标模型没有对应音区时会出现电子感。
- 咬字模糊：源人声分离质量和训练录音覆盖都会影响结果。

优先选择与目标模型性别、音域和声音厚度接近的引导轨。
