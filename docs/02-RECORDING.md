# 录音与数据集指南

## 推荐时长

| 有效录音 | 预期用途 |
|---|---|
| 1 至 3 分钟 | 只能快速验证，不适合作为正式歌手模型 |
| 5 至 10 分钟 | 可以获得基本音色，极端音区稳定性有限 |
| 20 至 30 分钟 | 推荐，能覆盖朗读、清唱和完整音域 |
| 60 分钟以上 | 边际收益较低，除非包含新的音区或唱法 |

## 可以分成多个文件

不需要把录音合成一个大文件。工作台支持一次上传多个文件，Applio 会自动切分。

推荐命名：

```text
01_daily_speech.wav
02_emotion_speech.wav
03_low_register.wav
04_mid_register.wav
05_high_register.wav
06_long_notes.wav
07_fast_song.wav
08_slow_song.wav
09_falsetto_breathy.wav
```

文件名可以使用中文，但模型名称建议仅使用英文字母、数字、下划线和短横线，例如 `my_voice_v1`。

## 内容比例

- 自然朗读：约 30%。
- 清唱歌曲片段：约 40%。
- 低、中、高音和长音：约 20%。
- 气声、真假声、轻重变化：约 10%。

## 录制标准

- 只录一个人的声音。
- 不播放伴奏，不添加混响和自动修音。
- 关闭风扇、空调、电视和手机提示音。
- 麦克风距离保持稳定，峰值不要削波。
- 使用 WAV 或无损格式优先，44.1kHz 或 48kHz 均可。
- 说错或唱错可以重录该句，偶尔音准偏差不会毁掉音色模型。

## 不要使用 AI 翻唱成品继续训练

换声成品包含分离残留、声码器误差和原歌手痕迹。反复训练会积累电子感。正式模型应优先使用本人原始干声。
